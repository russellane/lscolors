#### Synopsis
	lscolors sort [-h]

#### Description

Filter `stdin` to `stdout` sorting lines of a `DIR_COLORS` file by color then filetype. Blank
lines and comments are unsorted and moved to the end.

##### Optional Arguments
	  -h, --help  show this help message and exit
